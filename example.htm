This first example simply shows the "type ahead" functionality. Type ahead involves placing a single suggestion into the textbox as the user types. Only the part that the user didn't type is selected so that the user can continue to type and get appropriate suggestions.

Type a state name into the textbox below. Make sure the state name begins with a capital letter. The best example is to type "Missouri" one letter at a time. You will get suggestions of "Maine", "Michigan", "Mississippi", and then finally "Missouri".
